# Rare Agent Work: Fractional Autonomous Squads

## Objective
Build a revenue-generating web property at `www.rareagent.work` positioned as an elite provider of fractional, context-aware autonomous squads and agentic infrastructure. We sell the plumbing (memory, safety, orchestration), not the water (raw AI models).

## Core Offerings
1. **The Autonomous Ops Team ($5k-$10k/mo):** Leased multi-agent squads with persistent `memU` memory that integrate into a client's GitHub and Slack. They learn tech debt and institutional quirks over time, creating high retention.
2. **Agentic System Hardening (High-Ticket Integrations):** Consulting to install Idempotency Gates and Temporal workflow wrappers for enterprise clients building their own AI. We sell the safety rails.
3. **Hyper-Niche Legacy Integration:** Building translation layers between state-of-the-art AI and legacy systems (1990s ERPs, physical hardware, compliance pipelines) that generic models cannot navigate natively.
