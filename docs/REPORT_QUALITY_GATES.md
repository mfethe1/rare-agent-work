# Report Quality Gates (Paid Report Standard)

## Minimum pass criteria
1. Coverage: at least 40 unique high-quality sources
2. Recency: at least 70% of sources from last 14 days
3. Citation integrity: 100% of critical claims verified
4. Use-case depth: minimum 12 concrete use cases with implementation paths
5. Forecast quality: 2-week and 2-month scenarios with assumptions and risks
6. Value score: >= 80/100 by Brutal Value Critic
7. Owner approval: required before publish

## Rejection reasons (log required)
- unsupported_claim
- stale_source_profile
- low_novelty
- low_actionability
- weak_forecast_basis
- poor_cost_to_value_ratio

## Report size target
- Standard: 50–70 pages
- Deep dive: 80–100 pages
